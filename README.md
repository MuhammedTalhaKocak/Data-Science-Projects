# Data-Science-Projects
Bu repository satış ve pazarlama odaklı veri bilimi projelerini içermektedir. EDA, zaman serisi analizleri, müşteri segmentasyonu, A/B testi, churn analizi ve satış tahmini projelerini kapsamaktadır.
## Kullanılan Teknolojiler
Python / Pandas / Seaborn / Matplotlib / Scikit-Learn / Plotly
## Projeler
1. Proje EDA >>>  Bu proje veri keşfi ve veri temizleme süreçlerini içermektedir.
2. Proje Time Series >>> Bu proje zamanlara göre satışların genel trendini bulmaya ve zamanlar arasında gizli örüntüleri bulmayı içermektedir.
3. Proje RFM >>> Bu proje müşterileri sıklık, yenilik ve kazandırdıkları paraya göre (RFM) gruplandırmayı ve bu gruplamalara göre önerileri içermektedir.
4. Proje CHURN >>> Bu proje müşterilerin özelliklerine göre şirketten ayrılıp ayrılmayacaklarını tahmin etmeye yöneliktir.
5. Proje A/B Test >>> İki farklı Reklam Kampanyası arasında istatistiksel bakımdan bir anlam bulunup bulunmadığını inceler.
6. Proje Sales Prediction >>> Aylık satış verilerine göre satışları tahmin etmeyi amaçlamaktadır. Satışların doğru tahmin edilmesi şirketin performansı açısından kritiktir. 
## Nasıl Çalıştırılır
1. Repoyu klonlayın: `git clone https://github.com/MuhammedTalhaKocak/Data-Science-Projects`
2. Gerekli kütüphaneleri yükleyin: `pip install pandas numpy matplotlib seaborn plotly scikit-learn xgboost`
3. İlgili proje klasörüne girin ve notebook'u Jupyter'da açın.
